Source Code Management(SCM) is synonymous with version control, it is software tool that programmers use to manage the source code. It tracks the modifications to a source code repository and helps deal with merge conflicts

Adv of Source Code management:

- Track changes over time by creating a historical record.
- To compare the older versions and even undo some changes to the 
  code base. 

Importance of Source Code Management

- Track changes 
- Synchronization 
- Backup and Restore
- Undoing
- Branching and Merging
- Identifying conflicts and preventing overwrites

Version control is the management of changes of documents, computer programs, large websites and any other collections of information
